# 📝 Daily Task Planner (ToDoListApp)

A modern, robust, and user-friendly To-Do List application built with **SwiftUI** and **Firebase**. This app helps users organize their daily routines, manage tasks efficiently, and stay productive with real-time cloud synchronization.

## 🚀 Key Features

- **User Authentication:** Secure Sign Up and Login system powered by Firebase Auth.
- **Task Management:** Create new tasks, mark them as completed, and delete tasks with a smooth "swipe-to-delete" gesture.
- **Real-time Sync:** All data is instantly synced across devices using Firebase Firestore.
- **Personalized Profile:** View user details and manage sessions with a dedicated Profile page and Logout functionality.
- **Custom Branding:** Features a custom-designed AppIcon and a professionally configured Launch Screen.
